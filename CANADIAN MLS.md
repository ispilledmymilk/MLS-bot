# **Quick Reference — Province/Territory Summary**

| Province | Primary Board(s) | Platform | DDF? | Key Note |
| :---- | :---- | :---- | :---- | :---- |
| Ontario | TRREB, ITSO, WECAR | PropTx, Matrix | Yes (partial) | 4 feeds; VOW for sold data |
| British Columbia | GVR, FVREB, CADREB, VREB | BridgeAPI (RESO) | Yes (shared pool) | Reciprocal 3-board agreement |
| Alberta | CREB, EREB, GPAAR | Pillar 9 / Matrix | Yes (shared pool) | 60-day min listing; 3-day entry |
| Quebec | QPAREB / Centris | Centris | NO | Per-brokerage auth required |
| Saskatchewan | SRA | SRA MLS | NO | Must access SRA directly |
| Manitoba | WinnipegREALTORS | Paragon / WinnipegREALTORS | Yes | New Real Estate Services Act (2022) |
| Nova Scotia | NSAR | Paragon | Yes | AI compliance automation; accessibility fields |
| New Brunswick | NBREA | Board MLS | Yes | Written consent required for dual agency |
| Newfoundland | NLREA | Board MLS / DDF | Yes | Limited local online presence |
| PEI | PEIARA | Board MLS / DDF | Yes | Smallest provincial market |
| Yukon | Territorial Gov't | DDF | Yes | Licensing via territorial government |
| NWT / Nunavut | Territorial Gov't | DDF | Yes | Very small markets |
