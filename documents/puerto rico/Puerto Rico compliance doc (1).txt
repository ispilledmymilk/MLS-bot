1. Regulatory Framework — Quick Reference
The table below defines the governing framework for each compliance area. Use this as the authoritative reference when determining which law applies to a specific platform decision.


Compliance Area
	Governing Authority
	Platform Obligation
	MLS feed access and display rules
	Stellar MLS via PRAR / NAR IDX Policy 7.58
	Execute IDX/VOW agreements. Apply all display, attribution, purge, and compensation filter rules.
	Agent license verification
	JECVEBR — PR Examining Board / P.R. Laws Ann. tit. 20 Ch. 99A
	Verify active license + $10,000 bond before granting listing access.
	Listing status and CCP compliance
	Stellar MLS Rules / NAR CCP Policy 8.0
	Monitor public marketing triggers. Enforce 5 business day Stellar MLS PR entry window.
	FinCEN AML screening
	FinCEN RRE Rule — effective March 1, 2026
	Screen and flag entity/trust buyers in non-financed transactions. Notario files — platform flags.
	Fair Housing — federal classes
	Fair Housing Act (FHA) / HUD 2023 AI Guidance
	Audit ranking algorithms for disparate impact. No steering on 7 federal protected classes.
	Fair Housing — PR additional classes
	Puerto Rico civil rights laws
	Extend algorithm audits to: political ideology, social condition, sexual orientation, source of income.
	ADA web accessibility
	Title III ADA / WCAG 2.1 Level AA
	Full accessibility in English and Spanish. Annual screen-reader audit.
	Email marketing
	CAN-SPAM Act
	Opt-out model. Unsubscribe within 10 business days. $53,088/email violation penalty.
	STR listing compliance
	PRTC (Tourism Company) / Act 272-2003
	Verify PRTC Innkeeper ID before publishing STR listing. Disclose 7% room tax.
	Data privacy
	FTC Act Section 5 + state laws for out-of-PR users
	No PR-specific privacy statute. Apply CCPA/VCDPA etc. for users in those states.
	Sold price disclosure
	Puerto Rico is full disclosure jurisdiction
	No non-disclosure restrictions. Sold prices available via VOW without limitation.
	iMapp / Data Co-op tools
	Stellar MLS rules
	These tools are for Florida properties only. Must be blocked for all PR-address listings.
